POST to https://jsonplaceholder.typicode.com/posts

Body

{
    "userId": 1,
    "id": null,
    "title": "foo",
    "body": "bar"
}

Headers

User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36
Accept: */*
Content-Type: application/json

Curl

curl -v -X POST 'https://jsonplaceholder.typicode.com/posts' -H 'User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36' -H 'Accept: */*' -H 'Content-Type: application/json' -d '{ "userId": 1, "id": null, "title": "foo", "body": "bar" }'
